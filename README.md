# python_project

Minimal Python project skeleton.

Quick start:

1. Create a virtual environment:

   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

2. Install dev deps (optional):

   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:

   ```bash
   python main.py
   ```

4. Run tests:

   ```bash
   pip install pytest
   pytest
   ```
